Overview

Experience

WordPress Developer

Jordan Crown Design, Jan. 2013 - Present

At Jordan Crown, we created whole-site redesigns for our clients by creating custom WordPress themes and plugins. As a developer, I took PSDs and Sketch files from the designers and built out the site (backend and frontend) using PHP, Javascript, HTML, and SASS/CSS.

I usually reported to my project manager and the designer, but I frequently communicated with clients directly to collect project requirements and discuss potential changes, updates, additions, and bug fixes.

My position involved not only translating design documents into web pages, but also working with clients to come up with creative solutions to their problems, whether they were updating public-facing web site elements, or improving site administration usability. I took pride in doing everything I could to make things easy to use and visually pleasing.

In addition to building web sites, about 4 years of my time at Jordan Crown also involved occasionally building HTML emails for a marketing client of ours. I took design assets from the client and created HTML newsletters that I later sent out through their Hubspot account at the requested times.

Many projects I worked on also required me to manage web servers, usually CentOS or Ubuntu systems, but a few Windows servers as well. I set up Apache virtual hosts, MySQL databases, and DNS, and did some light Bash scripting to automate some more menial processes.

Project source control was done with Git.

Skills

HTML and CSS/SASS: I've been building well-structured, mobile-friendly layouts for almost a decade, and using SASS for about half of that.

WordPress: I have 12 years of experience building custom WordPress themes and plugins.

PHP: My 12 years of WordPress development involved both frontend and backend, which taught me a lot about using PHP.

JavaScript: Most of my JavaScript experience is in making snazzy visual effects and custom Google maps.

Linux Web Server Administration: I have some experience updating and maintaining Linux web servers, mostly Ubuntu LAMP stacks. I use DigitalOcean for my personal web server.

Git: I have a basic knowledge of Git for version control on all my projects. I've used Bitbucket and Github.

MySQL: My experience with MySQL is mainly in the context of WordPress, but I have occasionally dipped into custom table setup for highly performant record searches with millions of rows of data.
