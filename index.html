WordPress/Web Developer

Jordan Crown Design, Jan. 2013 - March 2025

At Jordan Crown, we created whole-site redesigns for our clients by creating custom WordPress themes and plugins. As a developer, I took mockup files from the designers and built out the site (backend and frontend) using PHP, Javascript, HTML, and SASS/CSS, occasionally doing some database administration and server administration.

I created custom blocks for the Gutenberg block editor using React and ACF. I developed a plugin for easier management and creation of custom blocks.

I usually reported to the project manager, but I frequently communicated with clients directly to collect project requirements and discuss potential changes, updates, additions, and bug fixes.

My position involved not only translating design documents into web pages, but also working with clients to come up with creative solutions to their problems, whether they were updating public-facing web site elements, or improving site administration usability. I took pride in doing everything I could to make things easy to use and visually pleasing.

A lot of our sites used interactive maps built with the Google Maps API.

In addition to building web sites, I also built HTML emails. I took design assets from the client and created HTML newsletters that were later sent out through Hubspot.

Many projects I worked on also required me to manage web servers, mostly Linux but some Windows. I set up Apache virtual hosts, MySQL databases, and DNS, and did some light Bash scripting to automate some more menial processes.

Project source control was done with Git.
